Shadow Wizards

2024

Les Technos

Javascript

Node.js

Socket.io

  Shadow Wizards est un projet de Shoot-them-up multijoueur a été réalisé entièrement en Javascript / Node.js. Les joueurs peuvent choisir leur nom, personnage et arme (boules de feus, pics de glace avec effet de ralentissement, ou lasers). Ils ont la possibilité de rejoindre une partie ou d’en commencer une nouvelle. Le but est de survivre le plus longtemps possible face à des vagues d’ennemis de plus en plus puissantes. Des bonus de vie et de dégâts apparaissent aléatoirement lors de mort d’ennemis. Le jeu fonctionne via des échanges socket.io entre front-end et serveur qui gère le back-end. Les scores sont enregistrés et le classement global et visualisable par tous les joueurs.

API REST Pizzaland

2024

Les Technos

PostgreSQL

Java

Tomcat

Arhitecture Rest

  Pizzaland API est un projet d'API réalisé suivant une architecture REST et implémenté en Java sur le thème d'une pizzeria. Les ressources telles que les commandes enregistrées, les pizzas disponibles et leurs ingrédients, sont disponibles en détails. De nombreuses requêtes sont disponibles telles que l'ajout, suppression, modification de ressources en détails. Par exemple : modifier les ingrédients d'une pizza, modifier le prix d'un ingrédient, se répercutant sur les prix des pizzas et commandes, enregistrer et supprimer des commandes ou pizzas, etc... Le serveur est composé de DAOs et contrôlleurs, renvoyant les DTOs en Json. Les requêtes de méthode GET sont publiques, alors qu’un système d'authentification de type Authorization: Basic {token} est implémenté pour les autres, afin de réserver la manipulation de la base de données aux administrateurs.

Chasse Au Monstre

2023

Les Technos

Java

Java FX

Arhitecture MVC

  Ce projet est un jeu-vidéo 2d au tour par tour développé en Java / JavaFx. Chaque joueur incarne un monstre devant s’échapper d’un labyrinthe ou un chasseur devant trouver la position des monstres, jouant leurs coups sur la carte. Une partie peut se dérouler également avec un joueur avec le rôle de son choix face à l’ordinateur. L’application adopte une architecture MVC constituée d’un modèle et d’une vue pour chaque joueur. Le labyrinthe est généré procéduralement et les intelligences artificielles de l’ordinateur utilisent divers algorithmes, notamment A* pour celle du monstre. Ce projet a requis une analyse de cahier des charges en détails ainsi qu’une planification de tâches rigoureuse.

Gestionnaire de voyage scolaire

2023

Les Technos

Java

Java FX

  Le but de projet était de développer un gestionnaire de voyages scolaires en Java / JavaFx destiné aux organisateurs. Le principe de l’application est de répartir de manière optimisée des élèves invités chez des élèves hôtes d’un autre pays. Pour cela, les informations de chaque élève peuvent être saisies via un formulaire ou être importées automatiquement via un fichier CSV. L’utilisateur lance ensuite un appariement automatique entre deux listes d’élèves s'appuyant sur un graphe, cet appariement prend en compte toutes les contraintes (allergies, régime alimentaire, …) et toutes les préférences (âge, centres d’intérêts, …) de chaque élève ainsi que des éventuels impératifs de l’utilisateur. Les résultats sont visualisables en détails dans l’application et peuvent être ensuite sauvegardés et exportés au format CSV.

Escape Game

2023

Les Technos

HTML/CSS

Javascript

Gant

  Ce projet a consisté à développer un Escape Game virtuel en équipe en s'appuyant sur un diagramme de Gant et des réunions d'équipe hebdomadaires. Nous avons onceptualisé des énigmes variées ainsi que de développer un scénario et un univers captivant que nous avons implémentés entièrement en HTML/CSS/Javascript. Le jeu est destiné à des étudiants en informatique de première année, il simule un Shell Linux dans lequel le joueur doit recolter des informations et modifier des données en Bash durant les énigmes.
